### 2.x 系

- 2.0.0: SCJ の体制変革とナンバー制の変更があったことにより大幅改造が行われる。

  開発環境・実行環境等の向上を鑑みて Dart が選ばれた。これより、Discord bot ライブラリは`Nyxx`を用いることとなる。
